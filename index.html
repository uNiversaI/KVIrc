Features We Offers

Features that makes us special and different others are as follows.

Hand-crafted with Details

Full GUI Based configuration, Floatable windows, Built-in help browser, Themes support, Drag & Drop toolbar editing, Activity Notifier Pop-Up and Custom Activity Notifier Sounds.

Light and Fast

Modular architecture: unfrequently used features reside in external modules and are loaded only when necessary, Extensibility: a huge API for writing extension modules and UNICODE support.

Netwrk Support & Security

Proxy support (SOCKSv4, SOCKSv5, HTTP), Full IPv6 support (also for DCC transfers), SSL support (also over IPv6), Micryption support.

Sleek and Smooth IRC Features

Multi-server connection, Notify lists, Ignore lists, Registered users database, Lag meter, Configurable logging, Standard DCC CHAT and SEND (with resume) support, Extended DCC types: VOICE, RSEND, GET, RECV, DCC CHAT over SSL, DCC bandwidth limit (ability to set it "on the fly").

Satisfied & Happy Users

Built-in scripting language interpreter, scalars, arrays and hashes..., aliases, popups, events, scriptable toolbars, Perl support and Multiple Language support.

Looks Amazing on Devices

Supports Linux, Mac OS and Wiindows. Now you can use your favourite IRC client on multiple plateforms. KVIrc also supports avatar for users and http/https connections.

Common Questions

Everything you need to know before you get started

What is KVIrc?

A IRC Client which helps you connect to IRC servers all over the world using address of that server. The address maybe ip address of the server or a DNS namespace address.

Is KVIrc malware?

NO! It is not a malware. The source code is available on GitHub. Anyone can review or audit the code. In fact we appreciate people to do code reviews. After all it's a Open Source project.

Can I use KVIrc as a portable application?

Yes! During instalation choose a portable directory and on first setup choose ("All settings in shared program folder (portable)"). Here portable directory means some "usb" path to install or even dropbox. There are users running kvirc fully from dropbox, Windows users at least

What language are available?

We support many languages. The supported languages are PORTUGUESE (BRAZIL), PORTUGUESE,ITALIAN, RUSSIAN, ESPERANTO, GERMAN, UKRAINIAN and HUNGARIAN. If you want KVIrc in your language contact us we will be happy to help you.

Does KVIrc work on my system? (Windows10/8.1/8/7, OSX, ...)

Yes! This awesome client works on any veriation of Linux, Windows 7, 8, 8.1, 10 and OS X. Now you can use your favourite client every where...
